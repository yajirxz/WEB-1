Listas

Funcionamiento

Las listas son usadas para resumir informacion y hacerla más claras.
Las listas mejoran el SEO.
Existen del tipo ordenadas, no ordenadas y descriptivas.

Listas ordenadas

Emplean en la etiqueta <ol> y la subetiqueta <li> que significa list item o elemento de lista

Ejemplos: una lista de pasos a realizar

  1. Sacar la herramienta
  2. Sacar la refacción
  3. Quitar llanta en mal estado
  4. Colocar refacción
  5. Guardar herramienta

Listas no ordenadas

Emplean en la etiqueta <ul> y la subetiqueta <li> que significa list item o elemento de lista

Ejemplos: una lista de compras

  1. Mermelada
  2. Atún
  3. Queso
  4. Huevo
  5. Pan
