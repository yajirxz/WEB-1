Formularios

Introducción

Es un control creado con el tag FORM, el cual permite obtener información de entrada desde el usuario


Atributos importntes
  • Action: Establece la URL destino de los datos
  • Method: Establece el verbo del protocolo HTTP con el cual se enviará la información. Los más usados son GET y POST
  • Name: Asigna un nombre al formulario para ser recuperado del lado del servidor

Tipos de entrada

  • Texto (TEXT): Permite capturar texto en claro
  • Área de texto (TEXTAREA): Permite capturar texto de múltiple línea
  • Contrseña (PASSWORD): Permite capturar texto oculto
  • Correo (EMAIL): Permite capturar correos electrónicos validados
