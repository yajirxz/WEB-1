Tablas

Con los tags TABLE, TR, TH y TD se crean tablas que organizan información para resumirla.

  • TR: Permite declarar renglones (rows)
  • TH: Es par declarar cabeceras de columna.
  • TD: Para declarar los datos de las celdas.

Los atributos: border, caption, colspan y rowspan

  • Border: Establece un ancho de borde para la tabla
  • Caption: Establece un título arriba de la cabecera
  • Colspan: Permite que una celda ocupe varias colmnas
  • Rowspan: Permite que una celda ocupe varios renglones
